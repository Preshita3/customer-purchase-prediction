## 📌 Project Overview
The Customer Purchase Prediction System is a machine learning-based web application that predicts whether a customer is likely to purchase a product based on features such as age and estimated salary.

This project demonstrates the complete machine learning workflow, from data preprocessing and model training to evaluation and deployment as an interactive web application.

---

## 🎯 Objective
- Predict customer purchasing behavior  
- Build a classification model using machine learning  
- Deploy the model as an interactive web application  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- Streamlit  

---

## ⚙️ Features
- 📊 Predicts whether a customer will purchase or not  
- 🧠 Uses Logistic Regression for classification  
- ⚡ Real-time prediction based on user input  
- 🖥️ Interactive UI built using Streamlit  
- 🌐 Deployable as a live web app  

---

## 🧠 How It Works
1. Dataset is loaded and preprocessed  
2. Features (Age, Salary) and target (Purchased) are defined  
3. Data is split into training and testing sets  
4. Feature scaling is applied using StandardScaler  
5. Logistic Regression model is trained  
6. Model is evaluated using accuracy and classification metrics  
7. User inputs are taken through the web interface  
8. Model predicts purchase decision in real-time
